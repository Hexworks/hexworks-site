Zircon Examples

Here you will find examples which you can copy and paste to your IDE to see how specific features of Zircon work.

They are also part of the project so you can check that out from GitHub and take a look at the code here. If you are using Kotlin, the examples are here.

The recommended option is to use one of the skeleton projects which you can clone from GitHub. They are designed to get you started with Zircon.

The Java skeleton project can be found here. For Kotlin you can use this skeleton.
